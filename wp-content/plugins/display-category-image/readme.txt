Display Category Image on Product Category Archives.

== Description ==
Display Category Image on Product Category Archives. 
When listing WooCommerce Product Categories, you need to show an image for each category header.
Category Archive Page Upload Image Field.

== Installation ==
1. Upload the plugin files to the '/wp-content/plugins' directory, or install the plugin through the WordPress plugins screen directly.
2. Activate the plugin through the 'Plugins' screen in WordPress.
3. That's All. This plugin works automatically so don't need to configure anything.

Now you need to set category Image from '/Products/Categories/your-choosed-category' EDIT and set Thumbnail Image.

== Screenshots ==

1. Set category Image from '/Products/Categories/your-choosed-category' EDIT and set Thumbnail Image.
2. New image field with (upload/remove) buttons to allow you to edit category or taxonomy image in quick edit.

== Frequently Asked Questions ==
= Does this work for WordPress < 5.0 ? =
If you are using a version before 5.0, It'll work but recommended to use latest Wordpress and WooCommerce.

= Does this work only in the WooCommerce ? =
Yes, this only works with WooCommerce.

== Suggestions ==
